# previsao-tempo
Projeto que mostra o clima atual de São Paulo, utilizando a API da openweathermap.

Para executar o projeto basta abrir o index.

Ele vai mostrar o clima atual, incluindo velocidade do vento, temperatura e condições do céu.

Por enquanto não há nenhuma interação com a pagina, mas pretendo adicionar pesquisa por cidades e localização automatica.
